## Projeto: Jogos de Simulador

### Descrição
Este projeto web é um catálogo de jogos de simulação, permitindo que os usuários pesquisem por jogos de acordo com o título, descrição ou tags.

### Tecnologias Utilizadas
* **HTML:** Estrutura da página web.
* **CSS:** Estilização da página.
* **JavaScript:** Lógica da aplicação, incluindo a pesquisa e a interação com o usuário.

### Como usar
1. **Clone o repositório:**
   ```bash
   git clone [URL inválido removido]
   ```
2. **Abra o projeto em um editor de código:** Recomenda-se usar um editor como Visual Studio Code.
3. **Abra o arquivo `index.html` em um navegador web.**
4. **Digite o nome de um jogo no campo de pesquisa e clique em "Pesquisar".**

### Estrutura dos arquivos
* **index.html:** Arquivo principal da página web.
* **styles.css:** Arquivo de estilos CSS para customizar a aparência da página.
* **dados.js:** Arquivo contendo os dados dos jogos (pode ser substituído por uma API ou banco de dados).
* **app.js:** Arquivo contendo a lógica JavaScript da aplicação.

### Como funciona a pesquisa
A pesquisa é realizada em tempo real, à medida que o usuário digita no campo de pesquisa. A função `pesquisar()` em `app.js` percorre a lista de jogos, compara o texto digitado com o título, descrição e tags de cada jogo e exibe os resultados correspondentes na seção "resultados-pesquisa".

1. Fork este repositório.

3. Faça as suas alterações e commit.
4. Envie um pull request.
